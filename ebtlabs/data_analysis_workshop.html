Data Analysis Project Workshop — Excel & Python (3 sessions)

A compact, hands-on workshop for college students to complete a small data-analysis project using Excel and Python (pandas). Sessions delivered live via Google Meet.

Organized by EBT Labs

Workshop at-a-glance

SessionDateTime (IST)Focus
Session 1 — Day 1 TBD 10:00 — 13:00 IST Excel for data cleaning, exploration, pivot tables, basic charts
Session 2 — Day 2 TBD 10:00 — 13:00 IST Python (pandas) for data cleaning, EDA, and visualizations
Session 3 — Day 3 TBD 10:00 — 13:00 IST Project work, presentations, feedback & next steps

Learning outcomes

  • Use Excel to import, clean and prepare a dataset for analysis.
  • Perform exploratory data analysis and create effective visualizations in Excel and Python.
  • Write basic pandas scripts to transform and summarize data.
  • Complete and present a small data-analysis project by the end of the workshop.

Prerequisites & software

  • Laptop with recent Excel (or Office 365) installed.
  • Python 3.8+ with pandas, matplotlib; we will provide a starter notebook (or you can use Google Colab).
  • Google account to join Google Meet and access shared materials.

Materials provided

  • Workshop slides and sample datasets.
  • Starter Jupyter notebook & solution notebook.
  • Certificate of participation (optional — subject to organizer).

Assessment & project

Participants will form small groups (2–4) or work individually to complete a short data-analysis project using the supplied dataset. Each group will present a 5–7 minute demo during Session 3.
