Executive Summary

EBT Labs proposes a hands-on 1-week (3-day) Data Analysis Project Workshop using Excel and Python (pandas) for college students. The goal is to enable participants to clean, analyze and visualize data through practical exercises and a mini-project. Below we outline the structure, outcomes, and session plan for this workshop.

Workshop Scope & Format

  • 3 interactive sessions (3 hours each) conducted live via Google Meet.
  • Day 1: Excel for Data Cleaning, Exploration, and Visualization.
  • Day 2: Python (pandas) for Data Transformation and Analysis.
  • Day 3: Project Work, Presentations, and Feedback.
  • Certificates issued upon successful completion.

Learning Outcomes

Participants will learn to clean, analyze, and visualize real-world data using Excel and Python. They will complete a guided data project and present their findings in the final session.
